Um template de PowerPoint para criar imagens de prévia social do GitHub.

## Visão geral

Este template foi criado para imagens de prévia social do GitHub.

As imagens de prévia social do GitHub devem ser criadas com este tamanho:

```text
1280 x 640 px
```

Você pode personalizar o texto, as cores e o layout mantendo esse tamanho.

## Download:

[Download github_social_preview.pptx](https://github.com/europanite/github_social_preview_pptx_template/raw/main/github_social_preview.pptx)

```bash
curl -L -o github_social_preview.pptx \
  https://github.com/europanite/github_social_preview_pptx_template/raw/main/github_social_preview.pptx
```

Ou use `wget`:

```bash
wget -O github_social_preview.pptx \
  https://github.com/europanite/github_social_preview_pptx_template/raw/main/github_social_preview.pptx
```

### Requisitos
- Google Slides (importe o arquivo .pptx), ou
- Microsoft PowerPoint
